## 3. 依存脆弱性（npm audit）対応

| 対応 | 内容 |
|------|------|
| Next.js | `14.2.5` → **`14.2.35`**（14.x系の最新パッチ）に更新。多数の高/重大advisoryを解消。 |
| postcss | `overrides` で **`^8.5.15`** に固定（ネスト含む）。XSS advisory解消。 |

### 残存（意図的に保留）
- **Next.js の一部advisory**: 完全解消には `next@16`（メジャー破壊的変更）が必要。MVPの安定性を優先し、**next@16 への移行は別Issueに分離**（要回帰テスト）。本番影響度は低〜中（多くはセルフホスト構成や特定設定が前提）。
- **dev依存（vitest / esbuild / vite / eslint関連）**: テスト・ビルド用途のみで**本番バンドルに含まれない**ため、現時点では許容。

> フォローアップ: 「next@16 メジャーアップグレード」を別Issueとして起票推奨。

## 4. パフォーマンス方針（design.md §5.2 / requirements §5.2）
- チャットはストリーミング表示で体感速度を確保（実装済み）。
- 静的ページは SSG/プリレンダリング（トップ・学ぶ・プロンプト集・安全ガイド）。
- 画像最適化・LCP≤2.5s は実画像導入時に計測（実キャプチャ導入時の宿題）。
