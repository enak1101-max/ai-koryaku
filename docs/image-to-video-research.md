# 「写真を動かす（画像→動画）」機能 — 調査・設計メモ

| 項目 | 値 |
|------|-----|
| 種別 | 機能の実現可能性メモ（将来検討） |
| 作成日 | 2026-06-06 |
| 対象サイト | シニアからのAI入門編（ai-koryaku） |

---

## やさしいまとめ（最初に結論）

- **いまのサイトのままでは、写真を動かすことはできません。**（今のAIは「文章」を作るAIで、動画は作れない）
- でも、**別の専用AI**をつなげば**実現は可能**です。
- ただし注意点が2つ：**①お金がかかる（文章より高い）**、**②できあがるまで時間がかかる**。
- なので、**今すぐではなく「次のお楽しみ機能」**として考えるのがおすすめです。

---

## 1. なぜ今はできないのか

このサイトのAI（Claude）は、**言葉を読んで書くAI**です。
「写真を動画のように動かす」のは**まったく別の種類のAI**（動画を作るAI）で、別途つなぐ必要があります。

---

## 2. 実現するには何が必要か

「1枚の写真 → 短い動画」を作る専用サービスを組み込みます。
世の中には、写真をアニメのように動かすAIサービスがいくつかあります（例：Runway、Luma、Pika、Google Veo、Kling など）。
これらの「API（外部サービスへの接続口）」を使って実現します。

> ⚠️ 各サービスの細かい機能・料金は変わりやすいので、本気で進めるときは**その時点の最新情報を確認**します。

必要になる部品：

| 部品 | 何をする |
|------|----------|
| 写真をアップロード | 端末から写真を選んで送る・プレビュー表示 |
| 動画を作る部分 | 外部の動画AIに写真を送り、動画を受け取る（**数十秒〜数分**かかる） |
| 待ち画面 | 「作成中です…」とやさしく表示 |
| 料金の制御 | 1本あたりが高いので、回数を厳しく制限 |
| 安全・権利の配慮 | 人物写真の扱い、肖像権、不適切な使い方の防止 |

---

## 3. お金のこと（正直に）

- 動画づくりは、**文章を作るより「ずっと高い」**です。
- ざっくり言うと、**短い動画1本で、数十円〜数百円**かかることもあります（サービス・長さ・画質による）。
- 文章は1回1円以下だったので、**けた違いに高い**とイメージしてください。
- そのため、もし入れるなら **「有料プラン限定」** か **「1日1回まで」など強い制限** が前提になります。

---

## 4. 画面のイメージ（素案）

```
┌──────────────────────────────┐
│  写真を動かす                      │
│                                  │
│  [ 写真をえらぶ ]   ← タップで選択   │
│  （選んだ写真のプレビュー）          │
│                                  │
│  どう動かす？                      │
│  [ ゆっくり動く ] [ ズーム ] [ 手をふる ]│
│                                  │
│  ┌────────────────────────┐  │
│  │      動画をつくる          │  │
│  └────────────────────────┘  │
│                                  │
│  （作成中は「お待ちください…」表示）   │
└──────────────────────────────┘
```

シニア向けの工夫：
- 選択肢は**3つくらいまで**に絞る（迷わせない）
- 待ち時間は「コーヒーでも☕」のようにやさしく案内
- できた動画は**保存・共有**しやすく

---

## 5. おすすめの進め方

1. **今は入れない**（まずは無料の今の形で、使われるか様子を見る）
2. もし「やってみたい」となったら、**お試しで1つのサービスだけ**つないでみる
3. コストが高いので、**有料プラン限定の特別機能**として出すのが現実的
4. 進めるときは、**最新の料金とできること**を改めて調べてから決める

---

## 6. まとめ

- 技術的には**実現できる**。ただし**お金と時間**がかかる。
- このサイトの「簡単・安心」という良さとは相性が良い（写真が動くと喜ばれやすい）。
- **次のフェーズ（別の山）** として、必要になったときに着手するのが安全。
